# Mobile App

This folder is intended for the mobile version of the payroll/time tracking application.

## Suggested Architecture
- Framework: React Native, Expo, or Flutter
- Language: TypeScript
- Support: iOS, Android, and/or mobile web (PWA)

## Directory Layout
- `src/screens/` - Mobile screens for time entry, approvals, dashboard, and settings.
- `src/components/` - Mobile-specific UI components.
- `src/services/` - API clients, offline sync, and device services.
- `assets/` - Icons, images, and mobile assets.

## Next Steps
- Choose a mobile framework and initialize the project.
- Implement core employee workflows: clock-in/out, edit timecards, submit reports.
- Add offline data sync and geolocation verification for site-based work.
